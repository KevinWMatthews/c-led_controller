This project places test executables in the bin/ directory.

Run tests for a specific module using command line flags for individual CppUTest executables:
	-g <group>      contains group
	-sg <group>     matches group exactly
	-lg             print all group names
./bin/Test_<your_code> -sg <your_test_group>
